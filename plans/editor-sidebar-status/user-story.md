## US-01: サイドバーで各ファイルの公開ステータスを識別する

**ロール:** コンテンツ作成者（ターミナルを使わない）
**目的:** サイドバーを見るだけで、各ファイルの公開ステータスを把握したい
**価値:** ファイルを一つ一つ開かずに、どのファイルが未公開・更新あり・公開済みかを識別できる

### シナリオ 1: 未公開ファイルが公開済みファイルと視覚的に区別される

```
Given サイドバーに未公開のファイルと公開済みのファイルが両方存在する
When エディタのサイドバーを見る
Then 未公開のファイルのノードと公開済みのファイルのノードが、見た目で区別できる状態で表示されている
```

### シナリオ 2: 新規作成後に別ファイルへ移動してもどのファイルが未公開かわかる

```
Given 新規作成したファイル（未公開）が存在する
When 別のファイルをサイドバーから選んで開く
Then サイドバーに戻っても、新規作成したファイルのノードが公開済みのファイルと区別されて表示されており、どのファイルが未公開かを開かずに識別できる
```

### シナリオ 3: 更新ありファイルが未公開・公開済みと区別される

```
Given 一度公開済みのファイルを編集・保存した（更新ありの状態）
When エディタのサイドバーを見る
Then そのファイルのノードが「未公開」「公開済み」とは異なる状態として表示されている
```

### シナリオ 4: 公開済みファイルはステータスが変わっていないことがわかる

```
Given すでに公開済みで、かつ編集されていないファイルが存在する
When エディタのサイドバーを見る
Then そのファイルのノードが「公開済み」の状態として表示されている（未公開・更新ありとは区別される）
```
