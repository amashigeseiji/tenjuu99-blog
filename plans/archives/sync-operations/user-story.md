## US-01: 公開した記事を非公開に戻せる

**ロール:** ブログ執筆者
**目的:** 一度公開した記事を、サイトから取り下げたい
**価値:** 公開の判断をあとから変えられる。原稿を失わずに公開をやめられる

### シナリオ 1: 公開済みの記事を非公開にする

```
Given 公開済みの記事がある（手元とサイトの内容が一致している）
When 執筆者がその記事を非公開にする
Then 記事はサイトから取り除かれる
And 原稿は手元に残っている
And その記事の状態は未公開として表示される
```

### シナリオ 2: 非公開にした記事を再公開する

```
Given 一度公開して非公開に戻した記事が手元にある
When 執筆者がその記事を公開する
Then 記事は再びサイトに反映され、公開済みとして表示される
```

### シナリオ 3: 未公開の記事は非公開にできない

```
Given 手元にのみ存在する、一度も公開していない記事がある
When 執筆者がその記事を見る
Then 非公開にする操作は行えない（操作が提供されないか、行えない理由が知覚できる）
```

### シナリオ 4: リモートが参照不能なときは操作できない理由が伝わる

```
Given 公開手段が解決できない構成になっている（publish-abstraction の unknown 状態）
When 執筆者が記事を非公開にしようとする
Then 操作は実行されず、行えない理由が執筆者に知覚できる
And エディタの他の機能（サイドバー・編集・プレビュー）は動作を継続している
```

## US-02: 記事を削除できる

**ロール:** ブログ執筆者
**目的:** 記事を手元からもサイトからも消したい
**価値:** 不要になった記事を、残骸を残さずに消せる

### シナリオ 1: 公開済みの記事を削除する

```
Given 公開済みの記事がある
When 執筆者がその記事を削除する
Then 記事は手元のファイルシステムから消える
And サイトからも取り除かれる
And サイドバー等の一覧にも表示されなくなる
```

### シナリオ 2: 未公開の記事を削除する

```
Given 手元にのみ存在する未公開の記事がある
When 執筆者がその記事を削除する
Then 記事は手元から消える（サイトへの反映は発生しない）
```

### シナリオ 3: 削除をサイトへ反映する手段が失われない

```
Given 公開済みの記事の手元のファイルが、エディタを介さずにファイルシステムから削除されている
When 執筆者がエディタでその状態を見る
Then その記事がサイトには残っていることが執筆者に知覚できる
And サイトからも取り除く手段が失われていない
```

## US-03: リモートの内容を手元に取り込める

**ロール:** 複数のマシンで執筆するブログ執筆者
**目的:** リモートにある最新の内容を手元に取り込みたい
**価値:** 別のマシンで公開した記事の続きを、git を知らなくても手元で書ける

### シナリオ 1: 別のマシンで公開した記事を取り込む

```
Given リモートに、手元には存在しない公開済みの記事がある（別のマシンから公開された）
When 執筆者が取り込みを行う
Then その記事が手元に現れ、開いて執筆を続けられる
And その記事の状態は公開済みとして表示される
```

### シナリオ 2: 別のマシンで更新された記事を取り込む

```
Given 手元にもある公開済みの記事が、リモートではより新しい内容になっている
When 執筆者が取り込みを行う
Then 手元の記事が最新の内容になる
```

### シナリオ 3: 手元だけが進んでいる記事は取り込みで巻き戻らない

```
Given 手元の記事に、まだサイトへ反映していない変更がある（手元のほうが進んでいる）
When 執筆者が取り込みを行う
Then 手元の執筆内容は失われない（記事は巻き戻らない）
```

### シナリオ 4: 両側で変わっている記事では執筆内容が守られ、理由が伝わる

```
Given 手元とリモートの両方で、同じ記事が異なる内容に変わっている（どちらが新しいとは言えない）
When 執筆者が取り込みを行う
Then 手元の執筆内容は失われない
And その記事を取り込めなかったこと・その理由が、git の言葉（merge・コンフリクト等）を使わずに執筆者に知覚できる
```

### シナリオ 5: 削除した記事が取り込みで復活しない

```
Given 執筆者がエディタで記事を削除した
When その後に取り込みを行う（削除がサイトへ反映される前か後かを問わず）
Then 削除した記事は手元に復活しない（削除の意図が保たれる）
```
