## US-01: テキスト入力中にプレビューをボタン操作なしで確認できる

**ロール:** コンテンツ作成者
**目的:** テキストを入力しながらリアルタイムでプレビューを確認したい
**価値:** 「書く」行為を中断せずにプレビューが確認できる

### シナリオ 1: テキスト入力後にプレビューが自動更新される

```
Given ファイルがエディタで開かれている
When テキストエリアにテキストを入力して、しばらく入力を止める
Then プレビューペインが更新され、入力内容が反映される（previewボタンは押していない）
```

### シナリオ 2: 連続入力中はプレビューが更新されず、入力が止まってから更新される

```
Given ファイルがエディタで開かれている
When テキストエリアに連続してテキストを素早く入力する
Then 入力中はプレビューは更新されず、入力が止まった後にプレビューが更新される
```

## US-02: 画像挿入後にプレビューでその場で確認できる

**ロール:** コンテンツ作成者
**目的:** 画像をドロップして挿入した直後にプレビューに反映されることを確認したい
**価値:** 画像が正しく表示されるか不安にならずに確認できる

### シナリオ 1: 画像をドロップするとプレビューが即時更新される

```
Given ファイルがエディタで開かれている
When 画像ファイルをテキストエリアにドロップする
Then テキストエリアに画像のMarkdown構文が挿入され、プレビューペインに画像が表示される
```
