## US-01: frontmatter で参照した画像の公開

**ロール:** ブラウザだけで記事を公開するコンテンツ作成者
**目的:** 記事を公開・更新したとき、その記事が参照しているすべての画像（本文・frontmatter）が記事と共に公開されてほしい
**価値:** OGP 画像などの frontmatter で指定した画像が公開ページで欠けることがなくなる

### シナリオ 1: frontmatter で画像を参照する記事を公開すると画像も公開される

```
Given frontmatter に画像パス（og_image 等）を指定した記事がある
And   その画像はローカルに存在する
When  その記事を公開する
Then  記事がリモートに公開される
And   frontmatter で参照した画像も共に公開される
```

### シナリオ 2: 本文と frontmatter の両方の参照が公開される

```
Given 本文で画像Aを参照し、frontmatter で画像Bを参照する記事がある
When  その記事を公開する
Then  画像Aと画像Bの両方が記事と共に公開される
```

### シナリオ 3: 記事の更新でも frontmatter 参照の画像が公開される

```
Given 公開済みの記事がある
When  frontmatter に画像パスを追加して記事の更新を公開する
Then  記事の更新がリモートに反映される
And   追加した画像も共に公開される
```

### シナリオ 4: 本文参照のみの記事の既存挙動が保たれる

```
Given 本文の画像記法でのみ画像を参照する記事がある
When  その記事を公開する
Then  従来どおり、参照した画像が記事と共に公開される
```

### シナリオ 5: 画像を参照しない frontmatter 値は公開対象に影響しない

```
Given frontmatter に画像パスではない値（タイトル・テンプレート名等）だけを持つ記事がある
When  その記事を公開する
Then  記事のみが公開され、余計なファイルが公開対象に含まれない
```
