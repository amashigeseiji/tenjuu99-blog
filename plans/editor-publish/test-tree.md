# テストツリー: editor-publish

**作成:** 2026-06-27

## できるのツリー

```
エディタは編集内容を公開できる [editor]
├── 公開ボタンはリクエストを送信してフィードバックを表示できる [editor]  ← ブラウザAPI依存: 手動確認のみ
└── 公開ハンドラーは公開対象をリモートに反映できる [editor]
    ├── 公開対象コレクターは現ファイルと参照画像を収集できる [editor]
    │   └── 画像参照抽出器はMarkdownから画像パスを抽出できる [editor]
    └── 変更反映器は公開済み状態を更新できる [editor]
```

## 利用仮説

- 使ったらこうなるはず: Publishボタンを押すとターミナル操作なしにGitHubへのpushが実行され、編集画面に結果（完了または失敗）が表示される
- 使ったらこうなるはず: push対象には現在編集中のファイルと、そのMarkdownから参照されるすべての画像が含まれる
- こうなったら外れ: Publishボタンを押したのにターミナルで手動操作が必要になる
- こうなったら外れ: 画像が含まれずMarkdownファイルだけがpushされる
- こうなったら外れ: 失敗時に何の表示もなく操作が無言で終わる
